# Game Theory Toolkit

A growing collection of Python projects that explore game theory, microeconomics, strategic decision-making, and competitive market behavior.

Each project combines rigorous economic theory with interactive Python implementations, symbolic mathematics, numerical optimization, and detailed educational explanations. The goal is to build a comprehensive toolkit that teaches, analyzes, and visualizes the major concepts of game theory through software.

# Project Descriptions

## 01 — Prisoner's Dilemma Analyzer

A complete educational walkthrough of the classic Prisoner's Dilemma using a two-firm oligopoly framework.

### Features

- Complete payoff matrix analysis
- Economic interpretation of all four outcomes
- Best-response analysis
- Strictly dominant strategy detection
- Pure-strategy Nash equilibrium identification
- Prisoner's Dilemma explanation
- Joint-profit versus individual incentives
- Automatically generated Markdown output

### Core Concepts

- Strategic decision-making
- Dominant strategies
- Best responses
- Nash equilibrium
- Cooperative vs. non-cooperative behavior
- Oligopoly competition

---

## 02 — Cournot Competition Analyzer

A dynamic Cournot competition model supporting two- and three-firm markets using symbolic calculus and numerical optimization.

### Features

- Supports 2 or 3 competing firms
- Custom demand functions
- Firm-specific marginal and fixed costs
- Symbolic profit-function construction (SymPy)
- First-order and second-order calculus derivations
- Automatic reaction-function generation
- Simultaneous equation solving
- Cournot-Nash equilibrium computation
- Corner-solution detection
- Best-response verification
- Unilateral deviation testing
- Monopoly vs. Cournot vs. Perfect Competition comparisons
- Automatic Markdown report generation
- Reaction-function visualization

### Core Concepts

- Cournot competition
- Quantity competition
- Profit maximization
- Partial derivatives
- First-order conditions
- Second-order conditions
- Reaction functions
- Simultaneous equations
- Nash equilibrium
- Comparative market structures
